My Pokedex

Tecnologias: Java | Spring Boot | React | Docker | MySQL

MyPokedex é uma aplicação web fullstack que permite aos usuários explorar, gerenciar e interagir com dados de Pokémon. O frontend foi desenvolvido com React, e o backend utiliza Spring Boot para fornecer uma API RESTful segura e escalável.

  • Autenticação e autorização de usuários (com JWT)
  • Visualizar e buscar Pokémon por nome ou ID
  • Cadastrar novos Pokémon na Pokédex
  • Atualizar e remover Pokémon
Status: Em desenvolvimento

My Personal

Tecnologias: React | Tailwindcss

O MyPersonal é um site desenvolvido com React e TailwindCSS, voltado para a organização e visualização de exercícios de musculação. O objetivo principal do projeto é oferecer uma interface simples, moderna e responsiva para praticantes de musculação consultarem e explorarem diferentes tipos de exercícios.

  • Listagem de exercícios por grupo muscular (peito, costas, pernas etc.)
  • Visualização de detalhes do exercício (nome, imagem e descrição)
  • Busca por nome de exercício (Em desenvolvimento)
  • Possibilidade de cadastro, edição e exclusão de exercícios (Em desenvolvimento)
Status: Em desenvolvimento
